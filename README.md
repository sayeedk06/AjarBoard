<h1>AjarBoard</h1>

<h2> Technologies: </h2>
<ul>
<li>Django Rest Framework</li>
<li>Next Js</li>
<li>React-Grid-Layout
<li>Docker</li>
<li>Mysql/Postgres</li>
<li>Redis</li>
</ul>
